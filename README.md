# Electoral
Electoral Bonds EDA — India (2019–2024) Analyzed 18,871+ transactions from India's Electoral Bonds dataset using Python, Pandas, Seaborn &amp; Matplotlib. Uncovered donor patterns, party-wise fund distribution, and temporal trends through 6+ visualizations.


🛠️ Tech Stack
Python | Pandas | NumPy | Matplotlib | Seaborn

📂 Dataset

electroral_bonds_buyer_data.csv — donor name, date, denomination
rec_data.csv — political party, date of encashment, amount received


🔍 What I Did

Cleaned & preprocessed 18,871 transactions
Analyzed 1,316 unique donors & party-wise fund distribution
Built 6+ visualizations — pie, bar, area & horizontal bar charts
Found temporal patterns across years, months & days


📊 Key Findings

60% of all bonds were at ₹1 Crore denomination
Top donor — Future Gaming & Hotel Services (₹1,208 Crore)
7.6% of purchased bonds were never encashed
